## Quick Start

```bash
# 1) Install dependencies
npm install

# 2) Run dev server
npm run dev

# 3) Build for production
npm run build
```

## Pages scaffolded

- `/` Landing
- `/login` Login
- `/blog` Blog list
- `/blog/:id` Blog detail
- `/membership` Membership
- `/courses` Courses list
- `/courses/:id` Course detail
- `/checkout` Checkout
- `/calendar` Course calendar
- `/meeting` Meeting
- `/search` Search
- `/literature` Library
- `*` NotFound
